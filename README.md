# Arkanoid

Тестовое задание на реализацию основной игровой механики игры Arkanoid.

Проект выполнен на JavaScript с использованием PixiJS.

## Запуск проекта

Установка зависимостей:

```bash
npm install
```

Запуск в режиме разработки:

```bash
npm run dev
```

## Используемые технологии

- JavaScript (ES6)
- PixiJS 8
- Vite

## Реализовано

- главное меню и экран паузы;
- начало новой игры;
- экран с предысторией (лором);
- три игровых уровня;
- управление платформой;
- движение мяча;
- столкновения мяча со стенами, платформой и кирпичами;
- отражение мяча в зависимости от точки попадания в платформу;
- разрушение кирпичей;
- переход между уровнями;
- система жизней;
- экран загрузки уровня;
- анимация попадания по укрепленным кирпичам;
- анимация разрушения платформы при проигрыше.

## Не реализовано

В текущую версию не вошли некоторые дополнительные механики оригинальной игры:

- модификаторы (бонусы), выпадающие из кирпичей;
- подвижные противники.

## Примечание

В рамках тестового задания была реализована основная игровая механика Arkanoid и базовая архитектура проекта. Структура проекта позволяет без существенных изменений добавлять новые игровые объекты, бонусы, противников и дополнительные уровни.
